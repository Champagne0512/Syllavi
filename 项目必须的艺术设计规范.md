1. 设计哲学：流动的秩序 (Fluid Order)
核心概念：课程表是死板的网格，但学习是流动的心流。设计要打破网格的束缚，用圆润、光影和空间感来对抗焦虑。
关键词：透气 (Breathable)、甚至 (Immersive)、微交互 (Micro-interaction)、超级椭圆 (Super-ellipse)。
2. 色彩体系 (Color Palette)
忘掉“科技蓝”和“微信绿”。我们要用低饱和度高级灰配合高亮荧光点缀。
模式 A：白昼 (Day - Ceramic & Air)
类似于陶瓷与空气的质感。
背景色: #F2F4F6 (冷调灰白，而非纯白)
卡片背景: rgba(255, 255, 255, 0.65) + backdrop-filter: blur(20px) (磨砂玻璃)
主文字: #2D3436 (深炭灰)
强调色 (课程块):
莫兰迪进阶版:
Sage Green (鼠尾草绿): #87A8A4
Dusty Lilac (灰紫): #BCA0BC
Warm Sand (暖沙): #E2C2A4
Muted Coral (哑光珊瑚): #E08E79
模式 B：暗夜 (Night - Cyber Zen)
不是纯黑，而是深邃的午夜蓝紫。
背景色: #0F172A (深蓝黑)
卡片背景: rgba(30, 41, 59, 0.7) (半透深蓝)
发光点缀: 使用 CSS box-shadow 制造霓虹辉光，例如 #818CF8 的微光。
3. 核心组件设计：颠覆传统的课程表
这是你项目的灵魂。千万不要画Excel表格线。
A. 布局 (The Layout)
无边框设计：彻底移除表格线。利用负空间 (Negative Space) 区分课程。
时间轴隐喻：左侧时间栏不要写 "8:00-9:40"，而是用一条极细的淡色贯穿线，当前时间点用一个呼吸的点（Breathing Dot）表示。
超级椭圆：所有的课程卡片圆角不要用 border-radius: 10px，要模仿 iOS 图标的曲率（Squircle）。在 CSS 中虽然很难完美实现，但可以用较大的圆角（如 16px-24px）来模拟柔和感。
B. 课程卡片 (The Class Card)
样式：不要做成实心色块。
艺术化处理：
左侧色条：卡片主体是极其淡的颜色（接近白色），左侧 4px 宽的高亮色条指示科目。
或者：使用微渐变（Linear Gradient），从左上到右下，透明度从 20% 过渡到 10%。
文字排版：
课程名：加粗，字号 32rpx。
教室/老师：极细字体，透明度 60%，字号 24rpx。
不要把所有字挤在中间，尝试左对齐，留出呼吸感。
4. 排版与字体 (Typography)
微信小程序默认字体是 PingFang SC，这很好，但我们需要调整字重 (Font Weight) 和 行高 (Line Height)。
数字艺术：课程表的时间、日期，一定要找一款宽体无衬线字体或者等宽代码字体（例如 DIN Alternate 或 JetBrains Mono 如果能引入的话，或者用系统自带的 WeChat Number）。让数字成为装饰的一部分。
大标题：页面顶部的“第X周”，字号要极大（比如 60rpx），极细（Font-weight 200），或者极粗（800），形成强烈的视觉冲击。
留白：标题和内容之间至少留出 40rpx 的间距。
5. 视觉特效与代码实现 (The "Sauce")
想要“高级”，全靠 CSS3 的特效。
A. 弥散背景 (Mesh Gradient Background)
不要用纯色背景，用几个模糊的彩色圆球在背景漂浮。
code
CSS
/* WXSS 示例 */
.background-blob {
  position: absolute;
  top: -10%;
  left: -10%;
  width: 500rpx;
  height: 500rpx;
  background: radial-gradient(circle, #BCA0BC 0%, transparent 70%);
  opacity: 0.4;
  filter: blur(60px); /* 关键：高斯模糊 */
  animation: float 10s infinite ease-in-out; /* 让它缓慢移动 */
  z-index: -1;
}
B. 玻璃拟态 (Glassmorphism Card)
用于你的底栏 TabBar 或者弹出的详情页。
code
CSS
.glass-card {
  background: rgba(255, 255, 255, 0.25);
  box-shadow: 0 8px 32px 0 rgba(31, 38, 135, 0.07);
  backdrop-filter: blur(12px); /* 核心：背景模糊 */
  -webkit-backdrop-filter: blur(12px);
  border-radius: 24rpx;
  border: 1px solid rgba(255, 255, 255, 0.18);
}
C. 只有选中时才出现的UI
极简主义：默认状态下，隐藏不必要的按钮。只有当手指按住屏幕或滑动时，功能按钮才浮现。
6. 交互设计 (Interaction & Motion)
学习通最让人难受的是点击时的生硬感。
果冻效果 (Spring Physics)：
点击课程卡片时，卡片不要瞬间变色，而是稍微缩小 (scale(0.95)) 然后弹回。
页面转场：
不要用默认的从右向左滑入。
尝试 Shared Element Transition（共享元素转场）：点击课程卡片，卡片本身放大并展开成为详情页，背景随之虚化。
触感反馈 (Haptics)：
利用微信小程序的 wx.vibrateShort({ type: 'light' })。每次切换周数、点击课程，都给一个极轻微的震动反馈。
7. 差异化功能点 (The "Art" Part)
为了让你的项目“不落俗套”，你需要加一点没用的、但很美的功能：
心情日签 (Mood Quote)：
在课表的空白处（比如没课的下午），显示一句极简的英文诗句或哲学短句，配上极小的衬线体。
天气感知：
如果外面下雨（调用天气API），小程序的背景出现隐约的雨滴滑落效果，或者色调变冷。
专注模式 (Zen Mode)：
学习通功能（资料、作业）通常很杂乱。设计一个“沉浸模式”，开启后，全屏变成黑色或米色，只显示一个倒计时和一个白噪音播放按钮（雨声、篝火声）。
总结：你的 Check List
拒绝 #000000 纯黑和 #FFFFFF 纯白。
拒绝 1px 实线边框，用阴影和色块区分层级。
拥抱 大圆角 (border-radius: 24rpx+)。
拥抱 弥散光影背景。
拥抱 留白，一个屏幕内不要塞超过 5 个主要信息群。
一句话总结风格：
像是 Notion 的秩序感 + iOS 16 的锁屏美学 + Headspace (冥想App) 的治愈色调。
