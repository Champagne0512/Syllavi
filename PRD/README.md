## 快速概览

### 项目定位
**基于 AI 视觉识别的极简主义学业助理**

用户只需拍照上传课程表/教务通知，AI 自动识别并结构化存储，结合待办管理、资料库、专注模式，打造"学术禅意"的沉浸式学习体验。

### 核心功能
1. **AI 课程表识别** - 拍照自动生成课程表
2. **智能日程总览** - 周/日/月多视图课程管理
3. **待办与考试管理** - 作业deadline和考试提醒
4. **校园资料库** - 文件管理 + AI划重点
5. **专注模式** - 番茄钟 + 专注统计

### 技术栈
- **前端**: 微信小程序 (WXML/WXSS/JS)
- **后端**: Supabase (PostgreSQL + Auth + Storage + Edge Functions)
- **AI服务**: 腾讯云OCR + DeepSeek/Minimax LLM

### 目标用户
中国大学生（18-24岁），追求高效学习工具，偏好简洁美观的设计风格

## 设计理念

### Academic Zen (学术禅意)
- **色彩**: Morandi莫兰迪配色（低饱和度、高级感）
- **布局**: 大量留白、卡片式设计
- **交互**: 触觉反馈、流畅动画
- **字体**: 系统字体栈，等宽字体用于数字

### 核心原则
1. **简约不简单** - 功能完整但界面简洁
2. **AI赋能** - 减少用户手动输入
3. **沉浸专注** - 减少干扰，提升效率
